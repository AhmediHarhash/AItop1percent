# Chapter 1 — Memory Fundamentals and the Four Horizons

Memory is the invisible infrastructure that separates AI systems that feel intelligent from AI systems that feel broken. Every production system that persists beyond a single request needs a disciplined approach to what gets remembered, where it lives, who controls it, and when it expires.
