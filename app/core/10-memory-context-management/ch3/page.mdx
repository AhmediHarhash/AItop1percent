# Chapter 3 — Memory for Agents and Multi-Step Systems

Agents that forget their own plans mid-execution are worse than useless. Multi-step systems need working memory for in-progress tasks, checkpoint memory for recovery, and shared memory for coordination. Agent memory architecture is what makes the difference between an agent that completes complex workflows and one that loops forever.
